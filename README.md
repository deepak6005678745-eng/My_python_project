## 💼 Featured Projects

### 1️⃣ Blue Invoice Generator (blue_invoice.py)
* **Description:** A complete system to manage clients and products, and generate polished invoices.
* **Features:**
    * Creates a `clients.json` and `products.json` database.
    * Adds/Views details of clients and products.
    * Generates an invoice and saves it as a `.txt` file (using UTF-8 encoding).
* **Concepts Used:** JSON File Handling, User Input, Data Structures (Dictionaries/Lists), Loops, File I/O.

### 2️⃣ Unlimited Invoice Generator (unlimited_invoice.py)
* **Description:** An early iteration focusing on the dynamic addition of items to an invoice list.
* **Features:**
    * Allows adding an unlimited number of items (product name, description, unit price, quantity).
    * Calculates totals automatically.
    * Outputs the simple invoice format.
* **Concepts Used:** Lists, Loops, Basic Arithmetic, Print Formatting.
